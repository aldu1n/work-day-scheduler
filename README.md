# Work Day Scheduler

## Description

This is a simple daily planner application that allows users to create and manage their daily schedule for standard business hours (9 am to 5 pm). The application displays of the current day, and month at the top of the scheduler, color-coded time blocks to indicate whether each block is in the past, present, or future. It also featuer the ability to enter events and save them in local storage, and then come back to check on them later, even after refreshing the page.

The site was made by using Bootstrap, Day.js and jQuery third-party APIs, and is made responsive for most screen sizes.
